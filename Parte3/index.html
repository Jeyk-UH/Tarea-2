Parte 2.2 Flexbox

Flex Container

flex-direction: Define la dirección en la que se colocan los elementos (nowrap, wrap, wrap-reverse)
flex-wrap: Indica si los elementos deben ajustarse a una nueva línea (nowrap, wrap, wrap-reverse)
flex-flow: Es una abreviación de flex-direction y flex-wrap
justify-content: Controla la alineación de los elementos en el eje principal (flex-start, flex-end,
center, space-between, space-around, space-evenly).
align-items: Alinea los elementos en el eje transversal (stretch, flex-start, flex-end, center, baseline).
align-content: Ajusta el espacio entre múltiples líneas de elementos (flex-start, flex-end,
center, space-between, space-around, space-evenly, stretch).

Flexbox

Es un modelo de diseño en CSS que permite distribuir y alinear elementos de manera eficiente en un contenedor
flexible. Se usa para crear diseños responsivos y acomodar elementos sin necesidad de usar flotantes o posiciones absolutas

order: Cambia el orden de aparición.
flex-grow: Determina cuánto puede crecer un elemento.
flex-shrink: Define si un elemento puede reducirse.
flex-basis: Establece el tamaño inicial del elemento.
flex: Abreviación de flex-grow, flex-shrink y flex-basis.
align-self: Ajusta la alineación de un elemento individualmente.
