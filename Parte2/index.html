Display

block: Hace que el elemento ocupe todo el ancho disponible y se coloque en una nueva línea
inline: Permite que el elemento se mantenga en la misma línea que otros elementos sin aceptar propiedades de ancho y alto.
inline-block: similar a inline pero permite definir ancho y alto

Overflow

visible: El contenido que excede el contenedor sigue siendo visible.
hidden: Oculta el contenido que se desborda sin permitir desplazamiento
scroll: Agrega barras de desplazamiento siempre, incluso si el contenido no se desborda
auto: Agrega barras de desplazamiento solo si el contenido se desborda

Visibility

visible: (por defecto): El elemento se muestra normalmente.
hidden: Oculta el elemento sin eliminar su espacio en el diseño.
collapse: Se aplica a filas y columnas de tablas, eliminando su espacio

Elemento Position

Static

Los elemenos con position static se colocan segun el flujo normal del documento

Relative

El elemento se posiciona en relación con su ubicación original. Puedes modificarlo con top, left ...

Absolute

El elemento se posiciona en relación con su contenedor padre con si no hay ninguno, respecto a la ventana del navegador

Fixed

El elemento se fija en la pantalla, sin importar el desplazamiento de la página
